行動通訊架構

📶1️⃣語音

  • 頻率範圍在850-900 MHz
  • NMT(Nordic Mobile Telephone)：歐洲使用最廣泛的行動通信標準之一，AM/FM
  • TACS(Total Access Communication System)：英國與歐洲行動通信標準之一，FM

1981年首先商用，是世界上第一個自動式行動電話系統

2️⃣語音與文字

  • 速度每秒9.6-14.4 kbps
  • 頻率範圍在850-1900 MHz
  • GSM（Global System for Mobile Communications）: 語音服務，短訊(SMS)，資料和漫遊，使用SIM卡對用戶身份進行鑑別，蜂巢架構
  • CDMA(Code Division Multiple Access)：允許多用戶共用同一頻率，不需要SIM卡，採用RUIM/CIM卡辨識用戶，侷限美國地區，競爭2G標準，但最終輸給GSM

3️⃣多媒體，傳輸音訊、視訊和上網

  • 速度2 Mbps
  • 頻率範圍在850-2100 MHz
  • UMTS(Universal Mobile Telecommunications System): GSM的演進版
  • HSPA(High Speed Packet Access): UMTS的演進版，通过更高階的調變技術和信道編碼提升頻譜效率

4️⃣高清視訊，自駕車

  • 速度 1 Gbps
  • 頻率範圍在700-3600 MHz
  • LTE(Long Term Evolution): 全IP，QoS，MIMO，FDD和TDD兩種工作模式
  • LTE-A（LTE-Advanced）：載波聚合
  • WiMAX：競爭4G標準，最終輸給LTE

5️⃣物聯網，VR

  • 速度 20 Gbps
  • 頻率範圍涵蓋低頻、中頻和高頻段

高資料速率、減少延遲、節省能源、降低成本、提高系統容量和大規模裝置連接。
