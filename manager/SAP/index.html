企業營運軟體霸主

2種應用開發方式

SAP Cloud Application Programming Model (CAP)

* 開發工具:  SAP Business Application Studio / VS Code
* 語言: JavaScript/TypeScript (Node.js) 與 Java
* 資料庫:  HANA， SQLite、PostgreSQL（開發階段）
* API: OData v4、REST、GraphQL
* 描述資料與服務模型: Core Data Services (CDS)
* 專為 SAP Business Technology Platform (BTP) 打造

ABAP RESTful Application Programming Model (RAP)

* 開發工具:  ABAP Development Tools (ADT, Eclipse)
* 語言: ABAP
* 資料庫: HANA 
* API: 自動生成 OData v4 服務
* CDS View、Behavior Definition/Implementation 定義商業邏輯
* 可部署於 S/4HANA（on-premise） 或 SAP BTP ABAP Environment（雲端）
* 強調企業級安全、事務一致性與 Fiori UI 整合

基礎建設

SAP BTP（Business Technology Platform）: = Infrastructure + Data + Integration + AI + Security

一套雲端的開發平台服務(PaaS),提供許多開箱即用的服務來滿足企業多個面向的資訊需求，用雲端技術（AI、資料分析、自動化、API、機器學習）去強化像 S/4HANA 這樣的核心系統。

CRM(Customer Relationship Management)是企業的對外窗口與前臺服務

負責管理企業與客戶之間的所有互動、銷售流程、行銷活動及服務支援，核心目標是提升客戶滿意度、建立客戶忠誠度，從而最大化營收和客戶價值。

ERP (Enterprise Resource Planning) 是企業的中樞神經系統

  • 負責整合財務、採購、生產、銷售、人事等資料與流程。
  • SAP是企業管理系統（ERP）的公司，在 SAP 世界中，這個核心實體就是 SAP S/4HANA。

S/4HANA
SAP 的第四代企業資源規劃系統，建立在 HANA 平台上的簡化智慧 ERP

R/1（1970年代）：Mainframe 上的單層架構系統。
R/2（1980年代）：支援多層架構與大型企業。
R/3（1990年代）：Client/Server 架構，成為全球 ERP 標準。
S/4HANA（2015年起）：以 HANA 資料庫為核心的第四代智慧 ERP，強調即時運算與雲端整合。
