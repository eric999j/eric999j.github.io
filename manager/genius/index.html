閱讀給下個資優生的7個秘訣

Posted on 2022-01-02 In 管理 Views:
Symbols count in article: 377 Reading time ≈ 1 mins.

在家中挖到國中生物跟班導，葉老師送的書，經過這麼多年，再看一次依舊受用，也發現現在很多處事方式就跟書中說的一樣，我很感謝她。

要點

  1. 把握學校教什麼，除了思考對方想知道什麼答案，還有這樣問的目的，為什麼我需要 ［跟40歲提到的一樣］
  2. 學習是件奢侈的事
  3. 用自己喜歡的方式 ［為了不學習疲乏］
  4. 當發覺遇到瓶頸要適時變換學習方式或內容 ［為了不學習疲乏］
  5. 製作學習設計圖［語言類別，OKR］
  6. 樹立自己短中長期人生目標
  7. 滿足好奇心

學習戰略

  1. 先專注理解，寫下的只能是腦中組織過的內容
  2. 回顧自己寫過的，再接著活用［定義，圖表，例題］
  3. 把不喜歡變喜歡
  4. 學習遷移，轉化困難問題
  5. 由易到難切割征服

秘訣

  1. 把文字化為圖像，比較好記
  2. 不要逃避，要找方法
  3. 誰，目的，如何準備
  4. 跟領域專家學習

家長

  1. 問孩子想要怎樣的人生
  2. 觀察孩子特質，讓他自己去闖
  3. 當孩子厭倦時提供轉化契機

讀完後發覺下次訂OKR前要寫條件，才知道自己有跟不足之處。
