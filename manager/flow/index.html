閱讀心流

Posted on 2020-10-04 Edited on 2020-10-31 In 管理 Views:
Symbols count in article: 445 Reading time ≈ 1 mins.

因單純想完成某件事，在執行中意識和諧有秩序而感到快樂富足的心理狀態，是一種禪，讓人拋下自我然後重新建立更大的自我，達成有挑戰的事不一定都是好的
身體心流在持續精進自己極限

本體焦慮/內在焦慮：缺乏內在秩序而焦慮的狀態
極度專注模式是自然將意識與書本文字交流時達成的狀態
事情可以分外在跟內在處理，外在安全感可以增加武裝，內在是知道無法避免，但求內心安定

樂趣8大要素

  1. 達成有挑戰但有機會成功的事
  2. 必須投注心力
  3. 有明確目標
  4. 有立即回饋
  5. 因為全心投入而忘卻煩惱
  6. 能掌控自己行動
  7. 忘卻時間流逝
  8. 享受這種體驗過程而且想再來一次

遊戲體驗探索創造

  1. 競爭型提升自身技能面對挑戰
  2. 投機型掌控不可知的幻覺
  3. 暈眩型改變現實認知帶來樂趣
  4. 模仿型忘卻自我

自成目標家庭

  1. 親子間目標與回饋明確
  2. 父母對自己在乎的事情也在乎
  3. 自己有選擇
  4. 親子關係的信任
  5. 放手讓孩子挑戰

培養抵抗挫折能力

  1. 不以自我為中心
  2. 對不確定事情開放樂觀
  3. 願意投入心力，找出路

心流模式

  1. 設立目標
  2. 全心投入
  3. 注意當下發生什麼
  4. 從善如流

感想

人生有了要一輩子做的目標就能走的有意義
而我一輩子的目標就是想幫助我接觸到的人
