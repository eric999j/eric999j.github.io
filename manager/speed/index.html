直接學習

  1. 訂學習計畫與明確產出
  2. 打造適合學習環境並沈浸其中
  3. 腦中預想學習過程模擬
  4. 不斷增加挑戰

反覆練習

  1. 將大目標分解成容易精進小目標
  2. 自我評估
  3. 模仿領域專家
  4. 找出學習會遇到的瓶頸，先解決
  5. 小目標有階段性，做不到下個就退回上一個
    試著提取一個尚未存在你腦中的答案，就像鋪設一條道路，可以帶領你走向一棟尚未被建造出來的建築物。

提取記憶

  1. 記憶卡問題答案配對背誦
  2. 自由回想在白紙寫下記憶
  3. 段落後寫重點簡單能回憶
  4. 自製挑戰方便去活用已知
  5. 描繪腦中輪廓確定已掌握

改善回饋

  1. 消除雜音，改進過程具體明確
  2. 持續修正到適合自己
  3. 組織問題關鍵點
  4. 擁抱能進步回饋並快速修正

預防遺忘

  1. 重複複習比一次要記全部更好，但間隔不宜過長
  2. 程序化培養成習慣
  3. 做出超乎預期的更佳表現
  4. 圖像記憶

培養直覺

  1. 還是要先面對複雜問題
  2. 從過程理解複雜問題
  3. 從具體可掌握的開始
  4. 寫下問題，想像自己如何處理，想像自己要傳授解法給他人

勇於實驗

  1. 學習方法，學習技巧，學習風格
  2. 先模仿再創造
  3. 比較各種方法
  4. 探索不同可能
  5. 混合不同技能
  6. 追求卓越
