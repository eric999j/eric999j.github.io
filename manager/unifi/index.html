Ubiquiti（UniFi）路由器

Posted on 2026-04-19 In 管理 Views:
Symbols count in article: 899 Reading time ≈ 1 mins.

新世代路由選擇

  • 網路設定、監控、升級、網路地圖，全在同一面板完成。
  • 大量設備、多人同時工作、高頻寬需求。
  • 可客用 Wi-Fi，區分NAS
  • 價位介於家用高階與傳統企業設備之間

#（UniFi）生態裡常被討論的幾款網關／路由器

Ubiquiti UniFi Dream Machine (UDM)

All-in-one：路由／防火牆／交換器／WiFi AP + 內建 UniFi Controller。硬體比舊世代強（較新 CPU、更多 RAM、儲存）

適合「想一步到位搞定有線 + 無線 + 基本網管」的家庭或小型辦公室。不需要額外 AP 或交換器就能覆蓋 WiFi。
需要擴充或自訂架構 (例如多 VLAN、分頻段、PoE 裝置) 時，就比較不彈性。

Ubiquiti UniFi Dream Router (UDR)

支援 WiFi 6、包含 PoE LAN 口、內建儲存（可用於 NVR／錄影等）及對 UniFi 生態（Network + Protect / Access / Talk 等 app）的支援

適合中小型家庭／辦公室，特別是未來可能擴充 WiFi AP、IP Camera、IoT 裝置時。性價比高。若網速 / 硬體需求不高，是性價比不錯方案。
如果你的網速很高、設備很多、常用 VPN / IDS／IPS，可能會成為瓶頸。

Ubiquiti UniFi UXG‑Pro Gateway ( UXG)

模組化 Gateway：只做「路由／閘道／防火牆」，不含 AP / Switch / Controller。本體通常性能與企業級更接近，適合搭配外部 Switch / AP / Cloud-hosted Controller。

適合已有 UniFi Controller 或 想自己管理 Switch / AP / 網路拓撲的「比較進階」場景。
你得自己搞 AP／Switch／Controller（或外接、雲端）

需求／場景

  • 家庭用，網速一般 (例如 < 500Mbps)，希望「插上就好，不想動太多設定」→ 建議選 UDR
  • 一次搞定「有線 + 無線 + Switch + 路由」 + 家用 + 想要內建 Controller，只想要穩定、好用的「all-in-one」→ UDM
  • 小型辦公室／有多間房、多個子網 (VLAN)、複雜網路需求／將來可能擴充很多 AP / Switch／設備→ UXG-Pro 更合適。
