15分鐘寫出爆紅千字文

Posted on 2022-11-05 In 管理 Views:
Symbols count in article: 653 Reading time ≈ 1 mins.

Google圖書正好有促銷，順便買來理解一下。

出書

標題

22-27字，篩選重要關鍵字，再簡單組合。
刪贅字，潤飾（重點數字）。
引人入勝的語句。

引言

150-200字，5w1h，一句人話訴說核心。
描述議題，強化重點，一段一個核心。

文章

刪贅字，慎選連接詞。
讓讀者有共鳴，考慮讀者選擇字詞。
在平凡文章製造矛盾，亮點，檢視時念出來。
長文章可以用小標題。

寫作訓練

  1. 觀察力（結合五感）
  2. 累積素材（新舊融合，正確資訊的判斷力）
  3. 聯想力（搭配個人特質，多角度的觀察，與自身相關的事最受注目）
  4. 小眾議題包裝成大眾議題（大眾：4-80歲）
  5. 組織（多角度銜接，正反面，太長做成專題）
  6. 流程（資料蒐集，採訪，客觀撰寫）
  7. 共鳴（檢視所寫，有無認同感）
  8. 稀釋（詳細描述單調的字：快樂-像小狗看到主人回家一樣）
  9. 模仿（大量閱讀，拆解元素，刻意使用）
  10. 比報（哪種呈現方式符合大眾，參考別人寫法）
  11. 字詞應用（異同，強弱，主詞，補充

三步找寫作主體

  1. 從自身專業，知識出發
  2. 跨界到其他主題[運用時事、專業、話題，來和自己的內容連結
    暖心，趣味，可愛]
  3. 我會？我喜歡？我是？

四個寫作階段

  1. 摸索：自我判斷
  2. 風格：磨練文筆
  3. 自信：個人風格
  4. 定位：決定領域

五個重要軟實力

  1. 資料彙整能力（重要［可看性］，正確［有公信力］，價值［有根據］，關聯［主軸，連貫］）
  2. 表達能力（個人特質，觀點）
  3. 邏輯思考能力（文章自我回顧，他人討論）
  4. 口語表達力（臨場反應，聯想）
  5. 企劃力（簡單話語感染人）

  1. Python
  2. 演算法
  3. 寫作能力
  4. 策略分析
  5. 人工神經網路

2020 在職進修最重視技能
