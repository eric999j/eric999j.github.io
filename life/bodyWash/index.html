洗髮精

Posted on 2023-12-16 Edited on 2024-04-18 In 生活 Views:
Symbols count in article: 473 Reading time ≈ 1 mins.

皮膚比較愛弱酸性。

差不多的配方，微調一下洗髮精就能變沐浴乳，洗手乳或果凍香皂。

潤絲精

  • 玻尿酸
  • 甘油

洗髮精/沐浴乳 750 ml

  • 25% 弱酸起泡劑 300 ml（40%，起泡效果較椰子油起泡劑佳，皮膚較不乾澀）
  • 或35% 椰子油起泡劑 90ml（12%）
  • 或70% 椰子油起泡劑 45ml（6%）
  • 甘油 50ml（頭髮不毛燥）
  • 75% 酒精 150ml
  • 全效清潔用品增稠劑 30ml/ 明膠：15g（1g 50ml）
  • 純水 220ml(35%:430ml, 70%:475ml)
  • DMDMH 2ml
  • 香精：適量
  1. （酊劑+增稠劑+水）加熱
  2. 再加入甘油+ 起泡劑+DMDMH攪拌
  3. 香精

感想

  • 配比時先以罐子容量與起泡劑濃度決定其他素材
  • 增稠劑可以再多一些


果凍香皂/洗手乳 237 ml

  • 25% 弱酸起泡劑 95ml（較椰子油親膚）
  • 甘油 5 ml
  • 75% 酒精 190ml
  • 清潔用品增稠劑：8ml /明膠：4g
  • 覆脂劑 1 ml
  • DMDMH 0.5ml
  • 香精：適量
  1. 將4g明膠倒入190ml的酊劑加熱融化
  2. 倒入95ml弱酸起泡劑
  3. 倒入5ml的甘油
  4. 添加剩餘配方，密封冰冰箱3-5小時

感想

  • 明膠也可以增稠
  • 不放冰箱會化掉

如果不放冰箱可以當增稠劑加入其他品項，或加水做成洗手乳
