IntelliJ IDEA 小技巧

Posted on 2023-04-15 Edited on 2023-04-28 In 工程 Views:
Symbols count in article: 481 Reading time ≈ 1 mins.

IntelliJ IDEA是款優秀的IDE，大大提升開發效率。

常用技巧

  1. In pom.xml, press ⌘ N, select Add dependency.
  2. control + enter to generate constructor.
  3. ⌥⇧⌘U generate UML diagrams.
  4. 游標放在被強調的程式⌥⏎查看修改建議
  5. ⌘⇧⌥U瀏覽 Class、Interface、Enum 及 Annotation 的關係圖
  6. ⇧⌘T快速在 Class 及 Test 間切換

Tips

  1. 代碼提示不區分大小寫
    Settings -> Editor -> General -> Code Completion
  2. 自動導包功能及相關優化功能
    Settings -> Editor -> General -> Auto Import

Dependency Injection

Unitest 3A

  • Arrange
    準備好測試環境、物件
  • Act
    使用測試資料執行目標函式
  • Assert
    比較測試結果跟預期結果是否一致
