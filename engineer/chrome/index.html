Posted on 2022-01-02 Edited on 2024-03-31 In 工程 Views:
Symbols count in article: 1.1k Reading time ≈ 1 mins.

這邊列出使用chrome based瀏覽器需要知道的資訊。

實驗功能

chrome://flags/

  1. Enable sharing page via OR Code: 網頁右鍵變QR Code分享
  2. Parallel downloading: 加快下載速度
  3. Desktop Screenshots，Desktop Sharing Hub in Omnibox，Desktop Sharing Hub in App Menu：開起來重啟網址列旁就自帶截圖
  4. TLS 1.3 hardening for local anchors: 增強安全，減少延遲

關閉分頁

如果使用的是 Windows 與 Linux，請按 Ctrl + w
如果使用的是 Mac，請按 ⌘ + w (關所有⌘ + shift+ w )

歷史圖表

chrome://discards/graph
