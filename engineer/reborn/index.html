舊硬體作業系統選擇

Posted on 2025-03-15 In 工程 Views:
Symbols count in article: 361 Reading time ≈ 1 mins.

\穢土轉生，再戰十年/

需求

  • 要可用Terminal，最好是Debian血緣
  • 低硬體要求
    • 古老的Asus N81Vg
    • 古早的Raspberry pi 3B+

候選OS

  • Lubuntu
    • 需200-300MB RAM
    • 輕量桌面，LXQt，自帶Nautilus的Samba的支持
    • 約20-25秒 啟動
    • 計算密集型效能: 中等
  • DietPi
    • 需50-100MB RAM
    • 主要為命令列，支援輕量Web GUI
    • 約10-15秒 啟動
    • 計算密集型效能: 優異
  • FydeOS
    • 約300-500MB RAM
    • Chromium瀏覽器、Android子系統支援
    • 約20-30秒 啟動
    • 計算密集型效能: 一般，偏向Web應用和多媒體優化

結論

  • Asus N81Vg > 用Lubuntu老硬體支援佳，有圖形介面
  • Raspberry pi 3B+ > DietPi，3B最佳化，當伺服器用

如果有更好硬體就可以試試FydeOS
